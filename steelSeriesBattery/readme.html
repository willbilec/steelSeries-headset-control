SteelSeries Battery Reader

This NVDA add-on reports the battery level of a compatible SteelSeries headset.

Usage

Press NVDA+Shift+A to hear the current battery level.

Notes

If another app is using the headset interface, NVDA may report that access is blocked.

The add-on includes HeadsetControl so it can read battery status directly on Windows without relying on SteelSeries GG services.
